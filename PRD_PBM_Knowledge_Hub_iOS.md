# Product Requirements Document: PBM Knowledge Hub (iOS App)

**Working Title:** PBM Knowledge Hub
**Author:** Manus AI
**Version:** 1.0
**Date:** November 11, 2025

---

## 1. Introduction

This document outlines the product requirements for the **PBM Knowledge Hub**, an iOS application that merges the comprehensive educational content from Neurofeedback Luxembourg's PBM resources with the practical HRV biofeedback tools from the "Coherence 365" project.

### 1.1. Project Vision

To create a unique, high-value application that serves as both a definitive educational resource on photobiomodulation and a practical tool for improving well-being through HRV biofeedback, positioning Neurofeedback Luxembourg as a leader in the field.

### 1.2. Target Audience

- **Primary:** Clients and potential clients of Neurofeedback Luxembourg.
- **Secondary:** Practitioners and individuals interested in learning about PBM and HRV.

---

## 2. Core Features

| ID | Feature | Priority | Description |
| :-- | :--- | :--- | :--- |
| F01 | **Educational Content Hub** | Must-have | A searchable library of all PBM and Vielight educational content, bundled for offline access. |
| F02 | **HRV & Symptom Tracking** | Must-have | Integration of the "Coherence 365" HRV biofeedback feature and a simple daily symptom tracker. |
| F03 | **Combined Dashboard** | Must-have | A unified dashboard to visualize the user's HRV session data and symptom scores. |
| F04 | **Offline Access** | Must-have | All educational content and user data to be stored locally on the device. |

---

## 3. Technical Specifications

- **Platform:** iOS (iPhone & iPad)
- **Language:** Swift
- **UI Framework:** SwiftUI
- **Data Storage:** Core Data or SwiftData for all content and user data.

---

## 4. GDPR & Data Privacy

- All user and content data will be stored locally on the device, ensuring maximum privacy and GDPR compliance.
- The app will not require an internet connection to function after the initial download.
